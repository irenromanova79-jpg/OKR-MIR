## 🎮 Правила игры

- На каждый вопрос **15 секунд**
- 4 варианта ответа — выбери правильный
- За правильный ответ начисляются **очки + бонус за скорость**
- Серия правильных ответов активирует **множитель x1.5 → x2 → x3**
- Всего **5 жизней** — старайся не ошибаться!
- После каждого ответа показывается **интересный факт** из учебника
